# Cloning the project 

```bash
    git clone https://github.com/almamunkhan09/Portfolio.git
```


## Installation

Go to the root directory of the project.Run the code below to install required packages

```bash
  yarn 
  
```
## Run in local enviornment 

```bash
    yarn dev

```
